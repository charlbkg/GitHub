Rough Resume Web Map Site


This site is a space for me to experiment building websites with the goal of making my own web-based GIS portfolio for the maps I've made in my undergrad.


I will also use this site to learn about incorporating web-mapping into sites I have developed myself. The end goal is to create a web-map that also functions as a resume, depicting the experiences I've had as points on a map that can be interacted with to provide more information about my skillsets and goals.


This links to the GIS portfolio I currently have on a pre-made google site.

Eventually, I will incorporate an embedded web map showing a spatial layout of the experiences listed on my resume. For now, I will link a GeoJSON file containing a subset of my experiences from Toronto. Creating this file helped me learn about creating the properties which will allow me to add information and context to my resume points.
